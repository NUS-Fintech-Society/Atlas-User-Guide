# Handling Users in Atlas

## Roles

Atlas contains 3 types of user roles.

| User Role | Privileges                                                                       |
| :-------- | :------------------------------------------------------------------------------- |
| Admin     | Able to create, view, edit and delete <b>Users, Events, Tasks and Applicants</b> |
| Member    | Able to view assigned tasks and events                                           |
| Applicant | Able to view interview status and tasks                                          |

## Updating User information

All users can update their own information by accessing the profile page.

You can edit the information of other users only if you are an admin. Simply click on the edit button next to the user's name on the dashbaord.
