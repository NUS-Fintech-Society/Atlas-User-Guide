# Creating a Single User

<Steps>
### Navigate to create users page

Go to /users/create by clicking on 'Create User(s)' button.

### Fill in user details

Fill in user details and check admin box to grant admin privileges.

### Default password is sent to new user

An email with a default password will be sent to the user's email. The user can login with the default password and is recommended to change the password immediately.

</Steps>
