# Task Management

Atlas task management is designed to allow department directors to assign and track tasks for department members quickly and efficiently. Once each individual has completed their part, they can mark the task as done.

<Callout emoji="⚠️">
  Only admins can create tasks for the entire department.
</Callout>

<Callout emoji="⚠️">
  Only when all members have completed the task, then the task will be marked as
  completed on the admin task dashboard.
</Callout>

## Creating a new Task

<Steps>
### Navigation

Click on the "Create Task" button on the Task Dashboard

### Fill in the Task Details

| Title            | Description                                                                                  |
| :--------------- | :------------------------------------------------------------------------------------------- |
| Task Name        | Enter a descriptive name for the task.                                                       |
| Task Description | Tell your members what you want them to do.                                                  |
| Department       | Select the relevant departments involved. These members will be the ones receiving the task. |
| Task Due Date    | Set the deadline for the task.                                                               |

</Steps>
